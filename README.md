# AI-Powered Clinical Decision Support System Using Reinforcement Learning

## Overview

This project builds an AI-based Clinical Decision Support System using Reinforcement Learning (Q-Learning). It generates synthetic patient cases, predicts disease, recommends treatment, suggests the correct doctor, and provides explanation with audio support using Gemini API and gTTS.

## Features

* Synthetic patient generation
* Q-Learning based treatment recommendation
* Doctor type suggestion
* Random vs Rule-Based vs RL policy comparison
* LLM explanation using Gemini API
* Audio explanation using gTTS


## Results

The RL model performed much better than Random Policy and close to the Rule-Based Policy.

**Final Result:**
RL ≈ Rule-Based >> Random
